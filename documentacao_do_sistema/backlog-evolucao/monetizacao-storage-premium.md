# 💰 Monetização de Storage Premium

## Descrição
Oferecer planos de expansão de storage para produtores que desejam manter galerias maiores ou feeds com centenas de fotos.

## Modelo
- **Plano Básico**: X fotos no feed.
- **Plano Pro**: 100 fotos + galeria de local estendida.
- **Plano Enterprise**: Storage ilimitado (ou limite muito alto) + CDN dedicada.
